# Contributing to N-Queens

## Build

Ensure Go 1.21 or later is installed.

```bash
go build
```

## Run

```bash
go run queens.go
```

Input N when prompted.

## Test

```bash
go test -v
```

All tests must pass before submission.

## Code Standards

- Follow Effective Go
- Use `gofmt` and `goimports`
- Explicit error handling
- Minimal, meaningful comments explaining why, not what
- All non-trivial code must have tests

## Submission

- One `.go` source file
- Clean, readable, well-structured code
- Passes all tests
- Complies with Go best practices
